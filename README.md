Navegar o moverse entre directorios:

PWD Command: 'print working directory', (mostrar ruta del directorio actual).

  ~/Directory$ pwd


Copiar vs Mover.

Copiar file:

  ~/Directory$ cp [file1.jpg] [le2.jpg] [file3.jpg] /home/square/Documents

  ~/Directory$ cp -r [file1.jpg] [le2.jpg] [file3.jpg] /home/square/Documents


Copiar Folder:

  ~/Directory$ cp [folder1.jpg] [folder2.jpg] [folder3.jpg] /home/square/Documents

  ~/Directory$ cp -r [folder1.jpg] [folder2.jpg] [folder3.jpg] /home/square/Documents


Mover file:

  ~/Directory$ mv [file1.jpg] [le2.jpg] [file3.jpg] /home/square/Documents

  ~/Directory$ mv -r [file1.jpg] [le2.jpg] [file3.jpg] /home/square/Documents


Mover Folder:

  ~/Directory$ mv [folder1.jpg] [folder2.jpg] [folder3.jpg] /home/square/Documents

  ~/Directory$ mv -r [folder1.jpg] [folder2.jpg] [folder3.jpg] /home/square/Documents


Eliminar un archivo o directorio(file or directory):

Eliminar archivo:

  ~/Directory$ rm -f [file_name]

Eliminar directorio:

  ~/Directory$ rm -r -f [directory_name]


Eliminar un archivo o directorio como administrador sudo(file or directory as sudo):

Eliminar directorio:

  ~/Directory$ sudo su

  user@user:/home/user/Desktop# ls

  user@user:/home/user/Desktop# ls -la

  user@user:/home/user/Desktop# rm -r -f [directory_name]

  user@user:/home/user/Desktop# exit

  ~/Directory$ sudo rm -r -f [directory_name]

Eliminar archivo:

  user@user:/home/user/Desktop# ls

  user@user:/home/user/Desktop# ls -la

  user@user:/home/user/Desktop# rm -f [file_name]

  user@user:/home/user/Desktop# exit

  ~/Directory$ rm -f [file_name]


Crear un archivo o directorio(file or directory):

Crear archivo:

  ~/Directory$ touch [file_name]

  ~/Directory$ touch /home/square/Documents/My-Folder

Crear directorio:

  ~/Directory$ mkdir [folder_name]

  ~/Directory$ mkdir [folder_name] /home/square/Documents/My-Folder


Ver el contenido de un archivo:

  ~/Directory$ cat [file_name]

  ~/Directory$ cat [OPTION] [FILE]

  ~/Directory$ cat /etc/issue

  ~/Directory$ cat /etc/issue /etc/lsb-release

  ~/Directory$ cat /root/linux > /root/linuxdistro
  
  ~/Directory$  cat file1.txt file2.txt > newfile.txt

  ~/Directory$ cat file1.txt file2.txt >> newfile3.txt

  ~/Directory$ etc...
